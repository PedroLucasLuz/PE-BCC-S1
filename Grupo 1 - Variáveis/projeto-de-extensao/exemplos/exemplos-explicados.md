# Exemplos Explicados

## Organização do Projeto

Os exemplos deste projeto estão separados por linguagem de programação, cada uma em sua própria pasta:

```text
📁 python
 ├── cadastro.py
 ├── dados.py
 └── jogos.py

📁 javascript
 ├── cadastro.js
 ├── dados.js
 └── jogos.js

📁 java
 ├── cadastro.java
 ├── dados.java
 └── jogos.java
 
```

Todos os exemplos possuem o mesmo objetivo: **demonstrar o uso de variáveis**, armazenando diferentes tipos de informações e exibindo seus valores na tela.

---

# Python

## cadastro.py

Este exemplo simula um cadastro de funcionário utilizando variáveis.

### Variáveis utilizadas

| Variável | Tipo                         | Descrição                          |
| -------- | ---------------------------- | ---------------------------------- |
| nome     | String                       | Nome do funcionário                |
| idade    | int                          | Idade do funcionário               |
| salario  | float                        | Salário do funcionário             |
| nivel    | char (String de 1 caractere) | Nível do funcionário               |
| ativo    | boolean                      | Indica se o funcionário está ativo |

### Objetivo

Demonstrar como armazenar informações de um funcionário utilizando variáveis de diferentes tipos.

---

## dados.py

### Variáveis utilizadas

| Variável  | Tipo    | Descrição                      |

# Conclusão

Os exemplos apresentados nas linguagens Python, JavaScript e Java possuem o mesmo propósito: demonstrar o conceito de variáveis.

As variáveis são utilizadas para armazenar informações na memória do programa, permitindo trabalhar com diferentes tipos de dados, como textos, números inteiros, números decimais, caracteres e valores lógicos (verdadeiro ou falso).

A organização em pastas facilita a comparação entre as linguagens e mostra como o conceito de variáveis é aplicado em cada uma delas.
