## Explicação:

1. ### EXEMPLO 1: 

É exemplificado como se adicionam itens na variáveis múltiplas. O "tupla.append(2)" está comentado, uma vez que as tuplas são imutáveis, sendo necessário delimitá-las previamente e, após isso, não é possível modificar suas propriedades, seja para adicionar ou alterar valores. 

2. ### EXEMPLO 2:

É exemplificado como se adiciona valores pelo índice usando o método “insert()” em uma lista.

3. ### EXEMPLO 3:

Exemplificação de uso do método “len()” para contagem de elementos.
