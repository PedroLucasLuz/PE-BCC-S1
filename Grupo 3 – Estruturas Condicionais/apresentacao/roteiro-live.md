## INTRODUÇÃO ÀS ESTRUTURAS CONDICIONAIS

A apresentação inicia explicando o que são estruturas condicionais. Elas são comandos utilizados na programação para permitir que o programa tome decisões de acordo com determinadas condições. Assim como acontece no dia a dia, um sistema pode seguir caminhos diferentes dependendo das informações recebidas. Para facilitar o entendimento, podem ser apresentados exemplos cotidianos, como decidir levar um guarda-chuva se estiver chovendo ou escolher uma roupa mais leve se estiver calor. Em seguida, mostrar que a programação utiliza a mesma lógica para executar ações diferentes conforme uma condição seja verdadeira ou falsa.

### IF

Após a introdução, apresentar a estrutura condicional if. Explicar que ela é utilizada quando desejamos executar um bloco de código apenas se uma determinada condição for verdadeira. Utilizar um exemplo do cotidiano, como: "Se eu tiver dinheiro, vou comprar um lanche". Nesse caso, a ação só acontece se a condição for atendida. Em seguida, mostrar um exemplo simples em Python, demonstrando a verificação de uma condição e a execução de um comando quando ela é verdadeira.

### ELSE

Na sequência, apresentar o else, explicando que ele funciona como uma alternativa ao if. Quando a condição do if não é satisfeita, o bloco do else é executado. Utilizar um exemplo do dia a dia, como: "Se estiver chovendo, vou ficar em casa; caso contrário, vou passear". Mostrar que existe sempre uma segunda possibilidade quando a condição principal não é atendida. Depois, apresentar um exemplo em Python contendo if e else para demonstrar a tomada de decisão entre dois caminhos possíveis.

### ELIF

Em seguida, explicar o elif, que permite verificar várias condições diferentes. Informar que ele é utilizado quando existem mais de duas possibilidades de decisão. Como exemplo cotidiano, apresentar uma situação relacionada a notas escolares: se a nota for maior ou igual a 7, o aluno está aprovado; se for maior ou igual a 5, está em recuperação; caso contrário, está reprovado. Mostrar então o exemplo correspondente em Python utilizando if, elif e else, evidenciando como o programa avalia cada condição até encontrar uma que seja verdadeira.

### MATCH CASE

Por fim, apresentar o match case, explicando que ele é uma estrutura utilizada para comparar uma variável com diferentes opções possíveis de forma mais organizada. Utilizar um exemplo do cotidiano, como a escolha de um dia da semana para exibir uma atividade específica. Mostrar que, em vez de utilizar vários elif, o match case permite organizar melhor as diferentes alternativas. Em seguida, exibir um exemplo em Python demonstrando a seleção de diferentes resultados de acordo com o valor informado.

## CONCLUSÃO

Finalizar reforçando que as estruturas condicionais são fundamentais para a tomada de decisões em programas. Destacar que o if é usado para verificar uma condição, o else oferece uma alternativa quando a condição não é atendida, o elif permite analisar múltiplas possibilidades e o match case organiza decisões baseadas em diferentes valores. Encerrar mostrando que essas estruturas estão presentes em diversos sistemas utilizados no dia a dia, tornando os programas mais inteligentes e interativos.
