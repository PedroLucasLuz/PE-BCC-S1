# Exercícios sobre estruturas condicionais

**1. Qual é a função das estruturas condicionais em um programa?**

- A) Repetir um bloco de código várias vezes.
- B) Armazenar informações na memória.
- C) Permitir que o programa tome decisões com base em condições.
- D) Criar variáveis automaticamente.

**2. Qual comando é usado para verificar uma condição em Python?**

- A) for
- B) while
- C) print
- D) if

**3. Quando o bloco else é executado?**

- A) Sempre que o programa inicia.
- B) Quando a condição do if é falsa.
- C) Quando a condição do if é verdadeira.
- D) Apenas quando existe um elif.

**4. Para que serve o comando elif?**

- A) Encerrar o programa.
- B) Criar uma variável.
- C) Repetir uma condição.
- D) Verificar uma nova condição caso a anterior seja falsa.

**5. Em uma estrutura if-elif-else, o que acontece quando uma condição é verdadeira?**

- A) Todas as condições são executadas.
- B) O programa gera erro.
- C) Apenas o bloco correspondente é executado e os demais são ignorados.
- D) O bloco else é executado junto.

**6. Qual das alternativas representa uma situação que pode ser resolvida com uma estrutura condicional?**

- A) Somar dois números.
- B) Decidir se uma pessoa pode votar com base na idade.
- C) Criar uma variável.
- D) Exibir uma mensagem fixa na tela.

**7. Em uma estrutura condicional, as decisões são tomadas com base em:**

- A) Comentários no código.
- B) Variáveis aleatórias.
- C) Condições avaliadas como verdadeiras ou falsas.
- D) Funções matemáticas.

**8. Por que as estruturas condicionais são importantes?**

- A) Porque substituem todas as outras estruturas da programação.
- B) Porque permitem que o programa execute ações diferentes conforme a situação.
- C) Porque eliminam a necessidade de variáveis.
- D) Porque servem apenas para cálculos matemáticos.

**9. O que o match-case permite fazer?**

- A) Comparar um valor com diferentes opções.
- B) Criar laços de repetição.
- C) Declarar funções.
- D) Armazenar dados.

**10. O match-case é mais indicado quando:**

- A) Há várias possibilidades para um mesmo valor.
- B) É necessário repetir um código muitas vezes.
- C) Não existem condições a serem verificadas.
- D) Deseja-se criar comentários.
