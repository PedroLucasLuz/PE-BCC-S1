# Estruturas Condicionais
### Estruturas condicionais permitem que programas tomem decisões, executando blocos de código diferentes dependendo de condições avaliadas como verdadeiras ou falsas.

## Nesse repositório você encontrará
* Slides explicativos sobreo tema
* Exemplos do conteúdo presente no nosso cotidiano
* Exercícios sobre o tema

## Objetivo do repositório

**Apresentar de forma simple e prática o conteúdo de estruturas condicionais, colocando em ação os conhecimentos desenvolvidos em sala.**
